Property-Level Coverage (Bank DSCR)

  • NOI (from NNN rent): equals base rent (tax/ins/cam pushed to tenant).
  • Debt Service (Property only): 1st (Seller @7%) + 2nd (Bank).
  • DSCR = Property NOI / Property Debt Service  →  target ≥ 1.25x
